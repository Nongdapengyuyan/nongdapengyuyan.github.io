搭建个人博客（借助nodejs+Git+Hexo)等技术

搭建博客

前言:

​ 几天前偶然间看到了舍友在搭建一个博客，就顺便询问了一些内容。正好之前自己也没尝试过自己搭建，就想着抽几天时间来搭建一个属于自己的博客。虽说在网上也有许多教程，最开始也感觉不太难，但是实际操作起来才知道这东西有多痛苦。前后搭建了将近一星期才完成（当然其中也不是天天在搞这个，所以实际上有个三四天时间吧）。在这我想把我搭建博客的完整过程记录一下，顺便也回顾一下搭建博客的经历。

工具准备

​ 首先，对于硬件方面我们肯定是要准备一台能够正常运行的电脑的。其次，软件方面，我们会用到Nodejs，Git，Hexo，GitHub等技术。

安装Nodejs(一个基于V8引擎的JavaScript 运行环境)

​ 大家可以选择直接到官网上下载自己所对应的电脑系统的版本即可:https://nodejs.org/en/ ,下载完后直接一直下一步默认安装，直到安装完成即可。

​ 安装完成后，Win+R 输入cmd 打开命令提示符窗口，输入”node -v”（中间有一空格），如果看到版本号，则说明安装配置成功。(如下图所示)

在GitHub创建仓库

​ 注册GitHub账号:https://github.com/ （国外网站，如果进不去的话可以多试几次或者尝试更换网络）

​ 注册好后点击右上角”+”号然后点击New repository创建仓库,仓库名必须为：你的GitHub用户名+github.io

同时，下面的权限设置也设置为Public权限，主分支名建议切换为master分支。

安装并且配置Git

​ 首先，我们先简单介绍一下Git：是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。也是Linus Torvalds为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件。

关于Git的详细知识会在后续文章中写道 。

Git官方下载地址：http://git-scm.com/downloads

官网慢，可以用国内的镜像：https://npm.taobao.org/mirrors/git-for-windows/。

Git的下载大家也可以去官网自行下载，下载完成之后也是一路点击Next默认安装即可。Git安装完后，我们要绑定个人的Github账号和进行一下配置的设置。在桌面上点击右键，随后点击Git Bash Here：

安装HEXO

​ HEXO就是我们的个人博客网站的框架， 这里需要自己在电脑里创建一个文件夹，可以命名为myblog（随意），Hexo框架与以后你自己发布的网页都在这个文件夹中。创建好后，进入文件夹中，右键打开GitBash或者在桌面打开cmd窗口，输入 cd 你创建的文件夹地址:

使用nmp命令安装Hexo，输入：(ps:安装时间过长，大家可以耐心等待)

npm install -g hexo-cli

安装完成后可以输入hexo -v查看，如果出现了一些版本号，就说明安装成功：

当然，如果大家一直尝试都安装失败的话可以尝试用国内的镜像资源，在这里给大家推荐一个安装cnpm的博客可以去看看：cnpm的安装(超级详细版) - uniapp - 博客园 (cnblogs.com)，输入的命令是:npm install -g cnpm -registry=https://registry.npm.taobao.org。

但是，安装完成后以后如果需要输入npm的地方都需要换成cnpm输入。安装完成后输入cnpm -v来查看是否安装成功:

保存之后，我们安装Git部署插件，进入生成的blog目录，右键打开GitBash或者打开cmd命令窗口随后切换到blog文件夹。输入如下命令：(提醒：如果之前安装好cnpm，这个时候就要把npm换成cnpm)

npm install hexo-deployer-git –save

部署好插件后，依次输入如下三条命令：

hexo clean
hexo g
hexo d

其中第三条的 hexo d 就是部署网站命令，d是deploy的缩写。正常情况下会弹出GitHub信息验证的窗口。

（图片转载于Hexo博客安装配置方法（Win版本） - soarli博客)

完成后，打开浏览器，在地址栏输入你的放置个人网站的仓库路径，即 http://xxxx.github.io你就会发现你的博客已经可以在网络上被访问了。（http://xxxx.github.io 进不去的话进 https://xxxx.github.io ）

写在最后

以上就是借助Hexo,Nodejs,Git等技术搭建的个人博客，本篇所写的仅是一些核心过程，在搭建博客的过程中我个人也遇到了各种各样的问题，在这里要感谢soarli的花园这位博客大佬，大家可以去看看他写的搭建博客的过程，我本人搭建的博客也还在优化过程中。最后，预祝大家在搭建博客的过程中一路顺利，没有bug！
