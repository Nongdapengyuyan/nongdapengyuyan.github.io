一，HTML简介

既然我们要学习一门编程语言，那我们就要先去了解它。

那么，什么是Html呢？

HTML的全称为超文本标记语言，是一种标记语言。它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。

超文本是一种组织信息的方式，它通过超级链接方法将文本中的文字、图表与其他信息媒体相关联。这些相互关联的信息媒体可能在同一文本中，也可能是其他文件，或是地理位置相距遥远的某台计算机上的文件。这种组织信息方式将分布在不同位置的信息资源用随机方式进行连接，为人们查找，检索信息提供方便。

所以，从内容上看，我们不难看出，Html实际上并不能算是一门编程语言，只能称得上是一种超文本标记语言。我们平时浏览的网页等等都是由Html等前端语言写出的。

二，基础内容

我们大概了解了一下什么是Html，所以接下来我们就该进入正题，去学习一下Html的基本内容了。

1.HTML基本标签

HTML标签

整个网页都是从<html>开始，到</html>结束的。

Head标签

head标签代表页面的“头”，定义一些特殊内容，这些内容往往都是“不可见内容”（在浏览器不可见）。

<head>内部标签 说明
<title> 定义网页的标题
<meta> 定义网页的基本信息（供搜索引擎）
<style> 定义CSS样式
<link> 链接外部CSS文件或脚本文件
<script> 定义脚本语言
<base> 定义页面所有链接的基础定位（用得很少）

Body标签

body标签代表页面的“身”，定义网页展示内容，这些内容往往都是“可见内容”（在浏览器可见）。

文本格式化标签

标签 语义 说明
<strong> strong(加强) 加粗
<em> emphasize(强调) 斜体
<cite> cite(引用) 斜体
<sup> superscripted(上标) 上标
<sub> superscripted(下标) 下标

网页特殊符号

网页特殊符号只需要记忆一个就行了，就是HTML空格“&nbsp;”，其他的特殊符号我们不需要记忆，当我们需要的时候再回来查找一下特殊符号表就OK了。

自闭合标签

HTML标签分为2种，一种是“一般标签”，另外一种是“自闭合标签”。一般标签有开始符号和结束符号，自闭合标签只有开始符号没有结束符号。

一般标签可以在开始符号和结束符号之间插入其他标签或文字。如：”<body><body/>”

自闭合标签由于没有结束符号，不能插入其他标签或文字，只能定义自身的属性。如：”<br/><hr/>”

HTML表格

表格由 <table> 标签来定义。每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 标签定义）。字母 td 指表格数据（table data），即数据单元格的内容。

数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等

表格语义记忆

表格基本标签
标签 语义 说明
table table（表格） 表格
tr table row（表格行） 行
td table data cell（表格单元格） 单元格
表格结构标签
标签 语义 说明
thead table head 表头
tbody table body 表身
tfoot table foot 表脚
th table header 表头单元格
