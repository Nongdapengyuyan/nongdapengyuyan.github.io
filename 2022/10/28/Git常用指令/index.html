Git常见指令

Git常见的六个命令:

1、push命令；2、pull命令；3、commit命令；4、add命令；5、checkout命令；6、fetch/clone命令。

Git有很多命令，如果想要完全记住是很难的，一般来说我们只要记住下图中的六个命令即可。

Git 命令清单及个别专用名词的译名如下：

Workspace：工作区

Index / Stage：暂存区

Repository：仓库区（或本地仓库）

Remote：远程仓库

生成SSH密钥:

ssh-keygen -t rsa -C "邮箱名"

查看SSH密钥:

cat ~/.ssh/id_rsa.pub

删除SSH密钥:

rm -rf ~/.ssh/*

一,新建代码仓库

在当前新目录新建一个仓库：

$ git init

克隆云端代码仓库：

$ git clone <url>

二,配置

显示当前的Git配置：

$ git config --list

设置提交代码时的用户信息:

$ git config --global user.name "用户名"

$ git config --global user.email "邮箱"

查看设置的用户名和邮箱：

$ git config user.name
$ git config user.email

删除邮箱和用户名:

$ git config --global --unset user.name "yourName"
$ git config --global --unset user.email "your@email.com"

三,增加/删除文件

添加文件到暂缓区:

$ git add [file1][file2]...

添加指定目录到暂缓区:

$ git add [dir]

添加所有目录文件到暂缓区:

$ git add .

删除工作区文件，并将这次删除放入暂缓区:

$ git add -p

四,代码提交

将文件从暂缓区添加到仓库区:

$ git commit -m "信息"

提交暂缓区指定文件到仓库区:

$git commit [file1][file2]... -m "信息"

提交工作区自上次commit之后的变化，直接到仓库区:

$ git commit -a

五,分支:

列出所有本地分支:

$ git branch

列出所有远程分支:

$ git branch -r

新建一个分支，但依然停留在当前分支:

$ git branch [branch-name]

新建一个分支，并切换到该分支:

$ git checkout -b [branch]

切换到指定分支，并更新工作区:

$ git checkout [branch-name]

版本回溯:

$git reset --hard 版本号

六,合并分支:

合并指定分支到当前分支:

$ git merge [branch]

删除分支:

$ git branch -d [branch-name]

七,查看信息:

显示有变更的文件:

$ git status

显示commit历史，以及每次commit发生变更的文件:

$ git log —stat

八,远程同步:

下载远程仓库的所有变动:

$ git fetch [remote]

显示所有远程仓库:

$ git remote -v

九,上传:

上传分支到远程仓库:

$ git push origin [分支名]
